# manchor

manchor 是一个macOS上运行的 anchor 工具，用于在屏幕右下角显示一个可自定义的悬浮时间窗口。  
支持透明模式、普通窗口模式、菜单栏控制、双击切换、右键菜单、窗口位置记忆等功能。

## 需求

1. 运行后在桌面右下角显示一个窗口
2. 运行后的窗口中显示当前时间，格式为 hh:mm:ss
3. 运行后的窗口背景为透明，窗口中显示的时间为半透明，时间的文本大小填充整个窗口，时间的颜色每秒随机切换，注意只更新文字颜色，不重绘整个窗口
4. 运行后的窗口总在最前显示，不可被选中，不可点击（点击穿透到窗口下方的应用），仿佛屏幕上的水印
5. 运行后的窗口在切换虚拟屏幕时保持显示在档期啊活跃的屏幕
6. 运行后在菜单栏显示一个图标
7. 双击图标后运行的窗口变成一个普通窗口，背景不再透明，可以被选中，可以被点击，可以调整大小，可以被拖动
8. 再次双击图标后，保持当前窗口的大小位置，使其恢复不可被选中，不可点击（点击穿透到窗口下方的应用）
9. 定义一个颜色列表，时间的颜色从列表中无序随机，但不可连续两次使用同一个颜色
10. 不要在dock中显示图标

## ✨ 功能特性

### **透明模式**
- 无边框、无交互、可穿透点击
- 悬浮在所有桌面与全屏应用上
- 文本颜色每秒随机变化

### **窗口模式**
- 可拖动、可调整大小
- 退出时自动保存窗口位置与大小，下次启动恢复

### **菜单栏图标**
- **左键双击**：切换透明/窗口模式
- **右键单击**：弹出菜单
  - **Lock / Unlock**（切换透明/窗口模式）
  - **Quit**（退出应用）
